מערכת לניהול למידה

AppsLearner היא מערכת לניהול למידה שפיתחתי עבור חברת AppsFlyer, המערכת מסננת ומציגה לכל לומד את תכני הלמידה המתאימים ביותר עבורו.
המערכת פותחה ב visual studio 2019, שפות הקוד והספריות הןhtml, javascript, css, c#, blazor, .NET : Web API, SQL
מערכת זו היא מחולל תכנים בעל שני צדדים, צד מנהל למידה וצד לומד. מנהל הלמידה מזין תכנים אל מאגר הלמידה. ניתן ליצור במערכת מספר בלתי מוגבל של מאגרי למידה, כל מאגר למידה יכול להיות בתחום שונה ועבור קהל יעד אחר. הלומד במערכת משיב על מספר שאלות ועל פי תשובותיו, יוצגו לו התכנים הרלוונטיים ביותר עבורו.
